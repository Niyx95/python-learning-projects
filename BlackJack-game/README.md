# Black Jack console game

# Description:
This is my BlackJack version made in python. simulates a game of BlackJack with his original rules.

# Learning goals:
- Iterating with lists
- calling list methods
- Practicing functions

# features:
- pick randomly 2 inital cards
- option to pick other card
- computer makes his choices
- Player closest to 21 wins

# How to run:

python main.py
