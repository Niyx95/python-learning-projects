This is a simple script helping me translate words from english fast. 

I build this to automate this process. I was genually bored starting a browser and have the entire process so i came out with this idea.I got this idea by reading How to automate boring things with python. 

Thanks for reading

How to use:
In working directory run ./main.py <word>

notes:
You need to have deep-translator and pyperclip installed. If not install them via pip.
