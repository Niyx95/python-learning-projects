# Proxy Server connection simulation

# Description
This small scripts tries to connect to a choosen proxy server.

# What I Learned
- Exception handling
- Handling user interruptions
- Time delays
- Building retry logic

# Features
- Choose from a proxy server list 
- Program tries to connect user to chosen proxy
- Program simulates connection delayed by 0.5 sec

# How to Run
```
python main.py
```
